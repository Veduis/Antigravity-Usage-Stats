# Changelog

All notable changes to the Antigravity Usage Stats project will be documented in this file.

---

## [2026-05-22] - Version 0.3.3

### 🚀 Antigravity 2.0 Compatibility
- **Protocol-Aware Discovery**: Enhanced connection layer to support both HTTP and HTTPS protocols dynamically per listening port.
- **Model Support Updates**: Added mappings for the latest Gemini 3.5 Flash, Gemini 3.1 Pro, and Claude 4.6 (Sonnet/Opus) models.
- **New Quota Pools**: Added dedicated visual indicators and emojis for the new `gemini-pool` (🪐) and `gpt-pool` (🧠) in the QuickPick dashboard.
- **Robust Fallback**: Updated connection handling to automatically probe and remember the correct communication channel.
